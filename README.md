#GenBI
🤖 Agentic GenBI Enterprise Platform
From Passive Dashboards to Autonomous Decision-Making
This repository contains a functional Proof-of-Concept (PoC) for a "To-Be" Agentic AI Architecture. Instead of requiring a human to manually monitor a Power BI dashboard, this platform uses an Autonomous AI Agent to audit data in Azure Synapse, reason over anomalies using LLMs, and provide proactive business recommendations via a Streamlit interface.
